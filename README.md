# Claude Code Plugins

A personal marketplace of [Claude Code](https://claude.com/claude-code) plugins.

## Installation

Add this marketplace in Claude Code:

```
/plugin marketplace add kartheekpenagamuri/plugins
```

Then install a plugin:

```
/plugin install resume-bullets@kartheek-plugins
```

## Plugins

### resume-bullets

Turns a career "brain dump" of projects, roles, and contributions into strong, impact-first resume bullet points — or reviews, critiques, and tightens resume bullets you already have.

Just describe what you've worked on (or paste your existing bullets) and the skill kicks in automatically. It:

- Leads every bullet with impact: `[result] by [what you did and how]`
- Quantifies wherever the source material allows — and never invents numbers
- Flags unrealized outcomes, ambiguous metrics, and inferred impact instead of overstating
- Writes for the skimming recruiter, not the domain insider
- Adapts emphasis for software engineers, program/product managers, and other roles
